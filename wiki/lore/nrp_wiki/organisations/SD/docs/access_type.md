# Классификация уровней доступа

Все блоки разделяются через **-**

В случае если необходимо указать несколько индексов одного блока, они устанавливаются через **:**.

Рядом с блоками с множественным выбором стоит соответствующая пометка.

!card

# Структура построения

|Наименование блока|Описание блока|
|------------------|--------------|
|Доступ информации с серверов АР|Данный блок указывает на максимальный уровень осведомлённости юнита об данных сервера АР (мисии, данные других юнитов и т.д.)|
|Доступ к аномалиям|Данный блок указывает на возможность юнита контактировать с аномалиями|
|Доступ к установке имплантов|[color=#FF0000][ОЖИДАЕТ РЕДАКТИРОВАНИЯ][/color]|
|Доступ к получению препаратов|[color=#FF0000][ОЖИДАЕТ РЕДАКТИРОВАНИЯ][/color]|
|Доступ к участию в исследованиях|[color=#FF0000][ОЖИДАЕТ РЕДАКТИРОВАНИЯ][/color]|
|Доступ к участию в миссиях|[color=#FF0000][ОЖИДАЕТ РЕДАКТИРОВАНИЯ][/color]|

!card_end
